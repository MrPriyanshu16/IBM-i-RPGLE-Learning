# Control Statements

## Overview

Control statements are used to control the flow of execution in an RPGLE program. They allow a program to make decisions, execute blocks of code repeatedly, and organize logic into manageable sections.

Modern free-format RPGLE provides structured control statements that improve readability and maintainability compared to traditional fixed-format coding.

---

## Topics Covered

### Conditional Statements

* IF
* ELSE
* ELSEIF
* ENDIF

Used to execute different blocks of code based on one or more conditions.

---

### Selection Statements

* SELECT
* WHEN
* OTHER
* ENDSL

Used when multiple conditions need to be evaluated, providing a cleaner alternative to multiple ELSEIF statements.

---

### Looping Statements

* DOW (Do While)
* DOU (Do Until)
* FOR
* ENDDO
* ENDFOR

Used to repeat a block of code until a specified condition is met.

---

### Loop Control

* LEAVE
* ITER

LEAVE exits the current loop immediately.

ITER skips the remaining statements of the current iteration and starts the next iteration.
